HARDWARE REQUIREMENTS

1. PC APPLICATION

2. WEB CAMERA

Applications are required which can scan the images or gesture movements and store data corresponding to the values.(for identifying the images.)

An External Camera can be used for capturing images while using devices with no built-in camera or while using desktop application.

3. Graphics Processing Unit(GPU)

4. Random Access Memory(RAM)



A Graphics Processing Unit(GPU) can be used while implementating deep learning models in order to speed up the training process.

Availability of sufficient ram plays an important role while performing machine learning algorithms and processing large captured images.
